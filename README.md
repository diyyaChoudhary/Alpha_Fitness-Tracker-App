# 🏋️ Fitness Tracker App

A simple and responsive **Fitness Tracker Web App** built using **HTML, CSS, and JavaScript**. This application helps users record their daily fitness activities, track workout duration and calories burned, and manage their progress using the browser's Local Storage.

## 🚀 Features

- ➕ Add fitness activities
- ⏱️ Track workout duration
- 🔥 Record calories burned
- 📋 View all saved activities
- 🗑️ Delete individual activities
- 💾 Data stored using Local Storage
- 📱 Responsive and user-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## ▶️ How to Run

1. Download or clone this repository.
2. Open the project folder.
3. Double-click `index.html` or open it using Live Server in VS Code.
4. Start adding your fitness activities.
